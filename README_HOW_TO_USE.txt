Welcome.

1. Go to your Git Clinet
2. File -> Clone repository.. and pick this repository from the organization
3. Save it where you want
4. Create a new project in Unity and open it
5. Copy the "Library" folder from the new unity project in the repository on the root
6. Open Unity and click on OPEN, pick the folder of your repository and you good to go


Update the repository

1. Go to your Git Clinet
2. Make sure your have the correct Current repository selected
3. Click on Fetch origin (or Pull origin)
4. Submit changes with a title
5. Fetch origin (Puch origin) again
